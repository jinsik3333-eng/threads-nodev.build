### Day 1 (4/27 일) — Claude Code 첫 만남
```
엑셀 함수도 어려워하던 제가, Claude Code랑 일주일째 같이 일하고 있어요 😅
"이거 자동화해줘" 한마디에 매주 3시간 걸리던 보고서 작업이 5분으로 줄었습니다.
코딩을 배운 게 아니라, 코딩하는 친구를 한 명 둔 느낌이랄까요.
비개발자 여러분, 진짜 해볼만해요 🙌
#비개발자 #ClaudeCode
```

### Day 2 (4/28 월) — 엑셀 자동화
```
매주 월요일 아침마다 전주 데이터 긁어모아 엑셀 정리하느라 커피 두 잔은 기본이었는데요.
Claude Code한테 "우리 회사 양식대로 주간 리포트 만들어줘" 했더니 진짜로 만들더라고요.
저는 그냥 숫자만 던져주면 끝나요. 이게 된다고...?
코딩 몰라도 됩니다. 시키기만 하면 돼요 💪
#업무자동화 #비개발자의삶
```

### Day 3 (4/29 화) — 개발팀 기다리기 vs 직접 만들기
```
개발팀에 "고객사별 견적서 자동 생성되면 좋겠다" 말했더니 백로그 3주 걸린다더라고요.
그날 저녁 Claude Code한테 똑같은 말 했는데 20분 만에 시연 가능한 데모가 나왔어요.
물론 실서비스 가려면 보완해야겠지만... 가능성을 봤다는 게 중요하잖아요?
비개발자도 이제 '기다리는' 사람이 아니라 '만드는' 사람이 될 수 있어요 🔥
#ClaudeCode #노코드아님직접만듦
```

### Day 4 (4/30 수) — 보고서 자동 생성
```
월말 보고서 쓰느라 야근하시는 분들 많죠? 저도 그랬어요.
이번 달은 Claude Code한테 "우리 회사 지난 6개월 보고서 스타일 학습해서 이번 달 초안 써줘" 했더니
핵심만 쏙쏙 뽑아서 깔끔하게 초안을 뽑아주더라고요. 퇴근 시간 지켰습니다.
AI가 내 일자리를 뺏는 게 아니라, 야근을 뺏어가는 거예요 😎
#월말보고서 #AI활용팁
```

### Day 5 (5/1 목) — 슬랙 알림봇 만들기
```
팀 단톡방에 매일 아침 오늘의 할 일 리마인드해주는 봇 있으면 좋겠다고 생각만 했는데요.
Claude Code한테 슬랙 API 붙여서 매일 9시에 투두 리스트 보내는 봇 만들어달라고 했더니...
진짜 만들었어요. 그것도 1시간 만에요.
이제 저는 우리 팀에서 '그 슬랙봇 만든 사람'입니다. 비개발자인데요 😏
#슬랙봇 #ClaudeCode
```

### Day 6 (5/2 금) — 코딩 두려움 극복
```
솔직히 말하면, 저는 개발자 콘솔만 켜도 손이 떨렸던 사람이에요.
그런데 Claude Code랑 작업하다 보니까 '코딩'이 아니라 '대화'라는 걸 깨달았어요.
"이거 에러났어" → "아 이 부분 때문이구나 수정할게"
마치 개발자 친구랑 카톡하는 느낌? 실수해도 괜찮고, 모르면 물어보면 돼요.
두려워하지 마세요. 우리는 코딩하는 게 아니라, AI랑 협업하는 거예요 🤝
#코딩입문 #비개발자
```

비개발자가 주말 프로젝트로 웹서비스를 배포하다니... 2026년 미쳤어요 🚀
#주말코딩 #ClaudeCode로만든거
