# @nodev.builder 리드 마그넷 로드맵 v2 — 2026-05-05

목적: 30일 콘텐츠 캘린더 v2에 맞춰 필요한 리드 마그넷을 다시 정리한다.

현재 방향:
- 설치/입문 콘텐츠가 늘어났다.
- Windows 사용자용 보조 자료가 필요해졌다.
- 가이드 4~6편, 7~9편을 중간중간 소개하므로 각 묶음을 받쳐줄 요약 리소스가 필요하다.
- 모든 게시글에 DM 키워드를 붙이지 않는다. 초기에는 프로필 링크/공개 리소스 허브 중심으로 운영한다.

---

## 1. 리드 마그넷 운영 원칙

### 만들 것

- HTML 1페이지 리소스
- 복붙 가능한 프롬프트
- 체크리스트
- 성공 화면 기준
- Mac/Windows 분리 안내
- 가이드 1~9편 로드맵

### 만들지 않을 것

- 긴 PDF
- 다운로드해야만 볼 수 있는 자료
- 너무 많은 키워드 DM 자동화
- 개발자 전용 설명
- 실제 화면 없이 텍스트만 긴 설치 가이드

### CTA 원칙

초기 계정에서는 키워드를 늘리지 않는다.

현재 유지:
- `시작`: 입문 3단계 가이드

나중에 추가 후보:
- `설치`: 설치 체크리스트
- `윈도우`: Windows 설치 막힘 해결
- `지시서`: Claude 업무 지시서
- `30일치`: 콘텐츠 캘린더 프롬프트

## 3. 제작 우선순위

### 0순위 — 이미 있고 계속 써야 하는 것

1. 입문 3단계 가이드 허브

### 1순위 — 이번 주 먼저 만들 것

1. Claude 설치/시작 체크리스트
2. Windows 설치 막힘 해결 체크리스트
3. 가이드 1~9편 로드맵

이유:
- 지금 캘린더의 초반 10일을 받쳐준다.
- 설치에서 막히는 사람을 바로 받을 수 있다.
- `시작` 키워드와 자연스럽게 연결된다.

### 2순위 — 다음 주 만들 것

4. Claude 업무 지시서 템플릿
5. 워크스페이스 구성 체크리스트
6. 연동 전 확인 체크리스트

이유:
- 가이드 4~6편 소개를 받쳐준다.
- 설치 이후 이탈을 줄인다.

### 3순위 — 3주차에 만들 것

7. Skills 반복 작업 템플릿
8. AI 역할 분담 템플릿
9. AI 작업 안전 체크리스트

이유:
- 가이드 7~9편 소개와 연결된다.
- 고급 기능을 비개발자 언어로 낮춰준다.

### 4순위 — 도달/전환용

10. 콘텐츠 30일치 Claude 프롬프트 키트

이유:
- 저장/공유/전환 가능성이 높다.
- 다만 현재 가장 시급한 문제는 설치/입문이므로 1순위에서는 내린다.
