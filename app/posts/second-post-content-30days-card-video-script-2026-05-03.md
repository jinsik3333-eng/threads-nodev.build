# 두 번째 게시글 + 15초 영상 스크립트 — 콘텐츠 30일치 Claude가 짜줌

## 연결 리소스

- HTML 키트: `/guides/resources/content-30days-prompt-kit.html`
- 공개 URL 예상: `https://jinsik3333-eng.github.io/threads-nodev.build/guides/resources/content-30days-prompt-kit.html`
- 운영 방식: 이 게시글에는 ManyChat 자동 DM을 새로 붙이지 않고, 프로필 링크/리소스 허브로 유도한다.

## 최종 카드뉴스 6장 구조

### 1장 — Cover

```text
콘텐츠 30일치
Claude가 짜줌
```

- 기존 trendy v8 인물/캠페인 톤 유지
- 얼굴을 가리던 우상단 프롬프트 박스는 축소해서 얼굴 밖으로 이동
- CTA: `프롬프트 복사 →`

### 2장 — Campaign bridge

```text
아이디어 하나가
30일 콘텐츠가 되는 과정
```

- HOOK / SLIDE / CAPTION 카드로 “하나의 아이디어가 콘텐츠 재료로 쪼개진다”는 느낌을 보여줌

### 3장 — Reframe

```text
감 말고
구조부터
```

- Claude에게 먼저 결과물의 형태를 정하게 함
- `/content-builder` 프롬프트 UI 노출
- 태그: HOOKS / VISUALS / CALENDAR

### 4장 — Output pieces

```text
Claude가 뽑은
콘텐츠 재료들
```

- 10개 훅
- 6장 구조
- 3가지 톤
- 30일 발행안

### 5장 — Workflow

```text
초안을
발행물로 바꾸는 법
```

- Claude → Notion → Canva → Instagram
- 프롬프트 → 캘린더 → 디자인 → 업로드

### 6장 — CTA

```text
프로필 링크에서
무료 키트
```

서브:
```text
프롬프트·캘린더·워크플로우를
프로필 링크에 묶어뒀어요.
```

버튼:
```text
프로필 링크에서 무료 키트 받기 →
```

## 캡션 초안

```text
콘텐츠를 매일 새로 생각하려고 하면 금방 멈춥니다.

문제는 아이디어가 없는 게 아니라,
하나의 아이디어를 여러 콘텐츠로 쪼개는 구조가 없는 경우가 많아요.

그래서 Claude에게 단순히
“콘텐츠 아이디어 줘”라고 묻지 말고,

후킹 / 캐러셀 구조 / 캡션 / CTA / 업로드 순서까지
한 번에 정리해달라고 시키는 게 훨씬 낫습니다.

복붙해서 쓸 수 있는 프롬프트 키트는
프로필 링크에 HTML로 열어둘게요.

PDF 자료집 말고,
바로 복사해서 오늘 써볼 수 있게 만들었습니다.

저장해두고, 주제 하나 넣어서 테스트해보세요.

#비개발dev #Claude #ChatGPT #AI업무 #콘텐츠마케팅 #인스타운영 #프롬프트
```

## 15초 릴스/쇼츠 스크립트

### 영상 제목

```text
주제 하나로 콘텐츠 30일치 뽑는 법
```

### 0~2초 — 결과 먼저

화면:
- 30일 콘텐츠 캘린더 표가 빠르게 보임
- 카드 5개가 펼쳐지는 모션

자막:
```text
주제 하나 넣었더니
30일치 콘텐츠가 나왔습니다
```

나레이션:
```text
콘텐츠 아이디어를 매일 새로 짜지 마세요.
```

### 2~5초 — 문제

화면:
- 빈 캘린더
- “오늘 뭐 올리지?” 메모

자막:
```text
문제는 아이디어 부족이 아니라
쪼개는 구조 부족
```

나레이션:
```text
대부분은 아이디어가 없는 게 아니라, 쪼개는 방식이 없는 겁니다.
```

### 5~9초 — Claude 입력

화면:
- Claude prompt window
- 프롬프트 핵심 줄 하이라이트

자막:
```text
Claude에게 이렇게 시킵니다
```

보이는 프롬프트:
```text
주제 하나를 30일치 콘텐츠로 쪼개줘.
제목, 포맷, 후킹, 캡션, CTA까지 정리해줘.
```

나레이션:
```text
Claude에게 아이디어가 아니라, 업로드 가능한 일정표를 요청하세요.
```

### 9~12초 — 결과

화면:
- hooks / carousel / caption / CTA / calendar 카드가 순서대로 등장

자막:
```text
후킹 · 구조 · 캡션 · CTA · 캘린더
```

나레이션:
```text
그러면 후킹, 구조, 캡션, CTA까지 한 번에 나옵니다.
```

### 12~15초 — 리소스 CTA

화면:
- HTML 리소스 페이지 첫 화면
- 복사 버튼 클릭 장면

자막:
```text
복붙용 프롬프트는
프로필 링크에 열어둘게요
```

나레이션:
```text
복붙용 프롬프트는 프로필 링크에 열어둘게요.
```

## 영상 제작 메모

- 처음부터 고퀄 애니메이션으로 만들지 말고, 화면 녹화 + 카드 이미지 + 자막으로 테스트한다.
- 첫 영상은 15초 이내.
- 텍스트는 화면마다 한 문장만.
- 카드뉴스와 같은 리소스 URL을 사용한다.
- 댓글 자동화는 붙이지 않고 프로필 링크 유도.
