## 2) 이미지 위 텍스트/스티커 운용

이미지 자체에 핵심 문구가 이미 들어가 있으므로, 업로드 화면에서 추가 텍스트는 얹지 않는 것을 권장.
대신 Instagram 본문에서 이모지로 시선 흐름을 만든다.

- 첫 줄: 문제 제기, 이모지 없이 차분하게 시작
- 중간: 🧭 로 `순서/가이드` 느낌
- CTA 앞: 📌 로 저장/안내 강조
- 마지막: 🛠️ 로 제작/실행 톤

스토리 공유 시에만 스티커 텍스트 사용:
- 1장 공유: `댓글에 시작 남기면 입문 순서 보내드려요 🧭`
- 6장 공유: `키워드: 시작`

## 3) ManyChat 자동화 설계

### 목적
댓글 키워드 `시작` → 입문 3단계 링크/안내 DM 발송.

### 트리거
- Instagram Comments
- 특정 게시물 선택
- 키워드: `시작`
- 유사 키워드 추가 권장: `시작!`, `시작해볼게요`, `가이드`, `입문`

### 공개 댓글 자동 답글
남겨주셔서 감사합니다. DM으로 입문 순서 보내드릴게요 🧭

### DM 1차 메시지
안녕하세요! @nodev.builder입니다.

처음 시작할 때는 전부 보려고 하기보다,
아래 3단계만 먼저 따라오면 충분해요.

1. 화면 감각 잡기
2. 도구 설치 따라하기
3. 첫 프로젝트 만들기

아래 버튼에서 입문 순서를 확인해보세요. 🛠️

버튼: 입문 3단계 보기
링크: [가이드 링크 입력]

### DM 2차 선택형 메시지, 1~3분 뒤
막히는 지점이 생기면
`이 화면에서 뭘 해야 하지?` 기준으로 다시 정리해드릴게요.

원하면 다음 게시물에서
초보자가 가장 많이 막히는 첫 화면부터 보여드릴게요.

버튼:
- 첫 화면부터 보기
- 나중에 볼게요

## 4) 운영 안전 규칙

- 첫날은 이 게시글 1개만 업로드
- ManyChat 자동 DM은 링크 1개만 포함
- 댓글 답글은 과하게 반복하지 않기
- 같은 날 추가 링크 게시물 올리지 않기
- Threads에는 투표형으로 가볍게 반응 수집
