# @nodev.builder 캐릭터 보드 브리프 — 2026-05-04

## 목적
첫 게시글 1페이지의 캐릭터를 앞으로 반복 사용할 수 있는 브랜드 마스코트로 정리한다. 2~6페이지에 무작정 장식을 넣기보다, 같은 캐릭터의 다양한 포즈/감정/상황을 필요한 곳에 조화롭게 넣기 위한 기준 보드를 먼저 만든다.

## 고정 요소
- 둥근 검정 뿔테 안경
- 검은 비니 또는 후드
- 다크 차콜 후드/워크웨어
- 작은 툴 파우치 또는 빌더 느낌의 소품
- 태블릿/노트북/워크플로우 노드 같은 추상 UI
- 따뜻한 베이지 스튜디오 배경
- muted teal, warm cream, charcoal, soft brown 팔레트

## 피해야 할 것
- 너무 아이 같은 비율
- 교육 앱/유아 콘텐츠 느낌
- 과한 표정/과한 몸짓
- 이미지 내부 텍스트/깨진 문자/로고
- 손가락 왜곡
- 각 포즈마다 얼굴/안경/의상이 달라지는 것

## 필요한 포즈
1. 기본 설명 포즈 — 태블릿을 들고 설명
2. 문제 발견 포즈 — 막힌 화면을 보고 생각함
3. 프롬프트 작성 포즈 — 노트북 앞에서 입력/수정
4. 체크/완성 포즈 — 체크 표시나 완료 보드를 들고 있음
5. 안내/CTA 포즈 — 손을 들어 “시작”을 안내하는 느낌, 텍스트 없음
6. 작은 반신/아이콘형 포즈 — 카드 구석에 넣기 좋은 작고 단순한 형태

## 사용 기준
- 1페이지: 큰 hero 캐릭터
- 2~5페이지: 정보 구조를 방해하지 않는 작은 보조 장면
- 6페이지: CTA를 방해하면 넣지 않음. 넣더라도 아주 작고 안정적으로.
