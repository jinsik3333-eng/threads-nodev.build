# 두 번째 게시글 image2 리셋 스토리보드

주제: 콘텐츠 30일치, Claude가 짜줌
계정: @nodev.builder
방식: image2 베이스 이미지 + 한국어 타이포 후처리. HTML UI로 슬라이드 전체를 만들지 않는다.

## 방향

- 레퍼런스: @mobileeditingclub + @adrianabubori
- 무드: 밝은 크림/스카이블루/화이트 기반의 프리미엄 캠페인 + Claude 워크플로우
- 금지: 긴 설명 박스, SaaS 랜딩페이지식 그리드, 툴 이름 나열, 얼굴 가리는 프롬프트 박스, 읽히는 가짜 UI 텍스트
- CTA: 새 계정 워밍업 때문에 ManyChat 키워드 추가 없이 프로필 링크 유도

## 6장 구조

### 01 Cover / Hook
- Role: 결과 약속을 1초 안에 보여준다.
- Message: 콘텐츠 30일치, Claude가 짜줌.
- Elements: 인물/노트북/플로팅 콘텐츠 카드, 넓은 좌측 여백.
- Form: 큰 SUIT 한글 타이포, `30일치`만 블루 강조, 작은 pill CTA.

### 02 Problem
- Role: 사용자가 겪는 막힘을 짚는다.
- Message: 매일 새로 생각하면 계정은 멈춘다.
- Elements: 빈 캘린더, 낙서 메모, 폰 초안 그리드.
- Form: 감성적인 빈 작업실 이미지 위 짧은 문제 카피.

### 03 Turn / Split
- Role: 해결 관점을 바꾼다.
- Message: 아이디어를 더 짜는 게 아니라 쪼개야 한다.
- Elements: 하나의 노트가 카드/캘린더/썸네일로 확장되는 장면.
- Form: 이미지 중심, 짧은 제목 + 3개 결과 라벨.

### 04 Claude Output
- Role: Claude에게 무엇을 시키는지 보여준다.
- Message: 결과물의 형태까지 먼저 정해주면 출력이 달라진다.
- Elements: 흐릿한 Claude-like 화면, 콘텐츠 캘린더, 폰 그리드.
- Form: 워크스페이스 이미지 + 짧은 작업 지시 카피.

### 05 Asset Board
- Role: 추상 아이디어가 발행 재료로 바뀐 결과를 보여준다.
- Message: 필요한 건 아이디어만이 아니라 바로 올릴 재료다.
- Elements: 무드보드, 캘린더, 체크리스트, 콘텐츠 조각.
- Form: 에디토리얼 콜라주 위 큰 메시지.

### 06 CTA / Resource
- Role: 저장/프로필 링크 행동으로 닫는다.
- Message: 복붙용 프롬프트 키트는 프로필 링크에 열어둔다.
- Elements: HTML 키트/프롬프트 카드/캘린더/폰 프리뷰.
- Form: 중앙 여백에 CTA, 하단 버튼. DM 유도 없음.
