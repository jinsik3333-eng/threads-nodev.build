# @gymcoding 분석 노트

## 역할
한국어 Claude/AI 개발 교육계정의 정보 위계·기술 번역·저장형 후킹 레퍼런스.

## 가져올 규칙
- 기술을 사용자 고민으로 번역
- 한 장 정리/5분 정리/3가지 패턴
- 흰색 제목 + 포인트 컬러 핵심어
- 상단 신뢰 라벨
- 저장 가치가 있는 내부 구성

## 버릴 규칙
- 유명 인물 사진 의존
- 뉴스형만 반복
- 밈/말풍선 과다
- 개발 교육 계정처럼 딱딱해지는 것

## 멀티슬라이드 확인 포인트
- 내부 슬라이드의 설명 밀도
- 저장형 체크리스트/요약의 위치
- 커버의 후킹과 본문 정보가 얼마나 연결되는지
