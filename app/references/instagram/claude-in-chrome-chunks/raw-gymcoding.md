### 게시글 2

- url: https://www.instagram.com/p/DX0lbjqk72b/
- cover_headline: 지금 쓰는 Claude API, 그거 옛날 코드일 수도?

- slide_roles:
  - S1: 커버 — "옛날 코드일 수도?" 의심 유발 훅
  - S2: Claude API 최신 변경사항 개요
  - S3: 변경점 1 — 구버전 코드 vs 신버전 코드 비교
  - S4: 변경점 2 — 새 파라미터/엔드포인트 안내
  - S5: 변경점 3 — deprecated된 기능 목록
  - S6: 마이그레이션 체크리스트
  - S7: 실제 코드 예시 (Before / After)
  - S8: 주의해야 할 에러 패턴
  - S9: 최신 상태 유지하는 방법 (공식 문서 링크 등)
  - S10: CTA — 저장/댓글 유도
- reusable_patterns:
  - "그거 옛날 거일 수도?" 의심 유발 훅: 독자가 자신의 현재 상태를 점검하게 만드는 불안 자극 패턴
  - Before/After 코드 비교 슬라이드: 텍스트 설명보다 코드 블록 대조가 개발자 타깃에게 직관적
  - 체크리스트 슬라이드: 저장 후 실제로 활용 가능한 형태로 구성해 저장 동기 부여
- avoid:
  - 코드 예시 없이 "업데이트됐어요" 텍스트만 있으면 개발자 타깃 설득 불가
  - 변경점 나열만 하고 "왜 바꿔야 하는지" 이유 없으면 행동 전환으로 이어지지 않음
