# 카드뉴스 색·톤 레퍼런스 리서치 v2

## 문제 진단
현재 Halfdone 계열 시안은 모바일 가독성은 좋아졌지만, 색과 톤이 다소 유치하고 강의/유튜브 썸네일 느낌이 강함.
- 블랙 + 형광 노랑 + 파랑 라벨 조합이 너무 직접적임
- 모든 요소가 강하게 강조되어 시각 피로가 있음
- 테크 정보형이지만 프리미엄/성숙한 무드가 부족함

## 조사 소스
- Behance: `instagram carousel design typography`
- Behance: `minimal instagram carousel serif`
- Behance: `editorial instagram carousel typography`
- 기존 사용 레퍼런스: @adrianabubori, @halfdoneclub.ai

## 후보 방향

### 1. Warm Minimal / Claude Editorial
참고 감각: Minimal Typography Carousel, LABONE 일부
- 배경: warm ivory / cream
- 텍스트: dark brown / near black
- 포인트: burnt orange / muted terracotta
- 그래픽: 큰 여백, 세리프 제목, 얇은 라인, 작은 UI 카드
- 장점: Claude 브랜드와 궁합이 좋고, 유치함이 가장 적음
- 단점: 후킹이 약해질 수 있어 제목 카피가 중요
- 팔레트: #F7F1E8, #211A16, #C46A3A, #E4C7A8, #FFFFFF

### 2. Clean Holographic System
참고 감각: LABONE
- 배경: white / light gray
- 포인트: 아주 연한 holographic mint/lavender/blue
- 그래픽: 문서 카드, 프롬프트 입력창, 시스템 플로우
- 장점: AI 제품/업무 시스템 느낌이 좋음
- 단점: 너무 깨끗하면 임팩트가 약할 수 있음
- 팔레트: #FFFFFF, #F5F5F2, #111827, #CBD5E1, #BFE9E0, #D8C8FF

### 3. Premium Dark AI
참고 감각: KIONA, Satr Brand Identity
- 배경: black / charcoal / deep navy
- 포인트: violet/blue 또는 amber를 극소량
- 그래픽: 미세한 글로우, UI 카드, 얇은 선, 넓은 여백
- 장점: AI/테크 전문성 강함
- 단점: 다시 어두운 톤으로 가되, 노랑/파랑을 줄이지 않으면 유치함 반복
- 팔레트: #09090B, #18181B, #F4F4F5, #A1A1AA, #8B5CF6 또는 #C97845

### 4. Black & White Editorial Typography
참고 감각: OFFWALL, Arabic Typography Experiments
- 배경: black or ivory
- 포인트: 거의 없음
- 그래픽: 텍스트 자체를 포스터처럼 사용
- 장점: 가장 성숙하고 디자인 감도 높음
- 단점: 한국어 카드뉴스에서는 딱딱하거나 난해할 수 있음
- 팔레트: #0A0A0A, #F4F0E8, #FFFFFF, #222222

### 5. Fashion Collage / Premium Magazine
참고 감각: Schiaparelli, Adriana 계열
- 배경: grayscale/photo collage/off-white
- 포인트: handwritten serif, thin line, editorial crop
- 장점: 유치함이 적고 프리미엄함
- 단점: Claude 업무/시스템 정보와 직접 연결하려면 이미지 소싱이 어려움
- 팔레트: #F3F0EA, #161616, #8B8178, #C7B9A5

### 6. Muted SaaS Workspace
참고 감각: Satr + Clean System 중간
- 배경: light slate / soft gray
- 포인트: muted blue / sage / graphite
- 그래픽: 실제 UI 스크린샷, 브라우저 프레임, 캘린더/문서 카드
- 장점: 업무 자동화, 비개발자, 생산성 콘텐츠에 가장 실용적
- 단점: 너무 SaaS 랜딩페이지 같아질 수 있음
- 팔레트: #F4F5F3, #1F2933, #68778D, #9AAE9A, #D9E2EC

## 현재 추천 순위
1. Warm Minimal / Claude Editorial
2. Clean Holographic System
3. Premium Dark AI
4. Muted SaaS Workspace

## 제외할 방향
- 형광 노랑 + 블루 라벨 + 블랙 배경을 그대로 유지하는 Halfdone 복제형
- 강한 네온 그라디언트
- 병원/부동산/마케팅 템플릿 같은 소셜 미디어 디자인
- 귀여운 파스텔/스티커/이모지 중심
- 모든 텍스트가 볼드인 정보 카드
