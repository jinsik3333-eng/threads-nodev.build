### L002: launchd vs Hermes cronjob
- launchd는 macOS 전용, 디버깅 어려움, 로그 분리
- Hermes cronjob은 크로스 플랫폼, 자연어 제어, 통합 로깅
- **결론**: 단순 반복 작업이 아닌 이상 Hermes cronjob 우선
