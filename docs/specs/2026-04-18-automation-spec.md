## 자동화 시스템 설계

### 기술 스택
- **언어:** Python
- **스케줄러:** macOS launchd (로컬 실행)
- **발행 API:** Threads 공식 API (Meta, 2024 공개)
  - 포스트 발행 / 댓글 읽기·쓰기 모두 지원
  - 우회 불필요

### 파이프라인 구조

```
[트렌드 수집] → [포스트 생성] → [예약 발행] → [댓글 모니터링] → [자동 대댓글]
```

#### 1. 트렌드 수집 (매일 06:00)
- Anthropic 블로그 RSS / Claude 릴리즈노트 크롤링
- 한국 AI 계정 Threads API로 최신 포스트 수집
- 요약 → 비개발자 언어로 변환 (Claude API)

#### 2. 포스트 생성
- 트렌드 요약 → 포스트 초안 생성 (Claude API)
- 경험 포스트: 운영자 아이디어 입력 시 수동 트리거

#### 3. 예약 발행
- launchd로 07:30 / 21:00 트리거
- Threads API로 발행

#### 4. 댓글 모니터링 (발행 후 2시간, 30분 간격)
- Threads API로 댓글 목록 조회

#### 5. 자동 대댓글
- **상위 10개 댓글** 자동 응답
- **키워드 트리거:** "자료", "어떻게", "저도요", "링크", "공유" 포함 댓글 우선 응답
- Claude API로 맥락 맞는 답변 생성

### 디렉토리 구조 (예상)
```
threads/
├── src/
│   ├── collector.py      # 트렌드 수집
│   ├── generator.py      # 포스트 생성
│   ├── publisher.py      # 발행
│   └── responder.py      # 댓글 대응
├── config/
│   └── settings.yaml     # API 키, 계정 설정
├── launchd/
│   └── nodev.build.plist # 스케줄러 설정
└── docs/
    └── specs/
```
